## Overview

* Module format: triple-width, full height
* Power consumption: 100mA

k2 is a version of the kastle2 (the latest desktop-device from [Bastl Instruments](https://bastl-instruments.com/)) especially tailored to the AE Modular format.

This module is a considerable step-up from the original Kastle devices ([still available from us](/modules/wonkystuff/kaestle.md)), with massively expanded sonic capabilities.

Existing firmware from the various Bastl webapps (FX-Wizard, Wavebard, and Alchemist) can be directly loaded into k2, and the open-source codebase allows you to even use the k2 as a platform for your own sonic explorations.
